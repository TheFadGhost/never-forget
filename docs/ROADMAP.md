# Roadmap

## Reliability

- Add device-specific reminder diagnostics and documented battery-optimization guidance.
- Expand exact-alarm, reboot, timezone-change, and daylight-saving test coverage.
- Add a reminder preview that shows the next scheduled notification times.

## Calendar Coverage

- Create a documented contribution format for new country and regional packs.
- Add more authoritative public-holiday packs with rule tests and source provenance.
- Improve `.ics` timezone support while preserving predictable offline behavior.

## Accessibility and Languages

- Complete a TalkBack and large-font audit.
- Add right-to-left layout verification.
- Extract user-facing text for community translations.

## Maintenance

- Publish signed release builds and reproducible release notes.
- Add compatibility tests for backup schema upgrades.
- Maintain a small set of contributor-friendly issues with clear acceptance criteria.
