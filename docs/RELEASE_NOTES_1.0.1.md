# Never Forget 1.0.1

The first private Android release of Never Forget.

## Highlights

- Start early: birthdays and important occasions can begin prompting weeks ahead.
- Stay visible: reminders escalate through the final days and remain actionable.
- UK + Bulgaria: regional UK bank holidays, Bulgarian public holidays, Orthodox Easter, and curated Name Days.
- Private by default: no account, ads, analytics, or backend.
- Built for ColorOS: in-app guidance for notification, exact-alarm, Auto launch, and battery settings.

## Installation

Download `NeverForget-1.0.1.apk`, allow installation from your browser or file manager when Android asks, then complete the four-step setup.

The APK requires Android 10 or newer.
