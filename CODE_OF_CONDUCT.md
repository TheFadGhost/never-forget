# Code of Conduct

Never Forget is a practical, welcoming project. Contributors and maintainers are expected to:

- Be respectful and constructive.
- Discuss technical decisions without personal attacks.
- Respect privacy, consent, and cultural differences.
- Avoid harassment, discrimination, threats, or publication of private information.
- Accept maintainer decisions about project scope and safety.

Unacceptable behavior may be removed from project spaces. Repeated or serious violations may result in a temporary or permanent ban.

Report conduct concerns privately through the repository owner's GitHub profile. Do not open a public issue containing sensitive details.
