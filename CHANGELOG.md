# Changelog

## 1.1.0 - 2026-06-12

### Added

- Standard `.ics` calendar import and export
- Duplicate-aware event import with per-event error reporting
- MIT license and public contribution, security, conduct, issue, and pull-request guidance
- Public roadmap for reliability, country packs, accessibility, translations, and maintenance

### Changed

- Prepared the project for a fresh-history public open-source release
- Documented the offline privacy model and contributor verification workflow

## 1.0.1 - 2026-06-08

### Added

- Native Android calendar and reminder application
- Persistent escalating reminder schedule with completion and snooze actions
- Daily reminder repair, stale-alarm cleanup, and immediate cancellation after completion
- UK regional holiday and occasion pack
- Bulgarian holiday, Orthodox Easter, and curated Name Day pack
- Alarm-ready country observances and followed Bulgarian Name Days
- Birthday ages, relationship labels, custom events, reminders, and tasks
- Contact/calendar import, encrypted backup/restore, and CSV transfer
- Home-screen widget, six themes, adaptive icon, and 120 Hz preference
- Automated unit, Room instrumentation, and Compose UI tests

### Notes

- Initial private release
- Optimized release build uses R8 code and resource shrinking
